# Customer_data_analysis
A data analytics project where customer transaction data was analyzed using Python for EDA and feature engineering. The processed dataset was visualized through a Power BI dashboard to uncover customer spending patterns, segment behaviors, and revenue drivers.

Project Goal
The goal of this project is to **simulate a corporate-grade, end-to-end data analytics workflow**, demonstrating the ability to translate raw data into **strategic business intelligence** by integrating Python, SQL, and Power BI.

Workflow Summary

1. Data Preparation, Modeling & Exploratory Data Analysis (Python)
- Performed using **Pandas**, **NumPy**, **Matplotlib**, and **Seaborn**.
- Cleaned and transformed raw customer data.
- Handled missing values, outliers, and created new features.
- Conducted **EDA** to uncover key customer behavior patterns.

2. Visualization & Insights (Power BI)
- Built an **interactive Power BI dashboard** highlighting:
  - Customer segmentation by age, region, and purchase frequency.
  - Sales and revenue trends over time.
  - Key performance metrics and actionable insights.
- Enabled stakeholders to explore **data-driven business decisions** visually.

3. Reporting & Presentation
- Compiled a concise project report summarizing major findings and recommendations.
- Designed a presentation that visually communicates:
  - Trends and customer insights.
  - Recommended actions to boost retention and revenue.
